To run this project locally:

1.  Clone this project locally
2.  Open a terminal 
3.  cd <project_directory>
4.  Run the requirements.txt file
    
    pip install -r requirements.txt
    
5.  Run main.py
    
    python3 main.py
   
6.  Click on the localhost link provided in the terminal
